# 设计规范 — AI 表情猜谜王

> 版本：v1.0 | 更新：2026-06-02

## 设计理念

**关键词**：年轻化、轻量感、emoji 文化、毛玻璃质感

深色背景 + 霓虹渐变点缀 + 毛玻璃卡片，营造"夜晚玩手机小游戏"的沉浸氛围。

## 配色方案

| 用途 | 色值 | CSS 变量 |
|------|------|----------|
| 背景色 | `#0f0f1a` | `--bg` |
| 卡片背景 | `rgba(255,255,255,0.06)` | `--card-bg` |
| 主文字 | `#e8e8f0` | `--text` |
| 次要文字 | `#9999aa` | `--text-secondary` |
| 主色调（按钮/高亮） | `#6c5ce7` | `--primary` |
| 主色调 hover | `#7d6ff0` | `--primary-hover` |
| 成功（答对） | `#00d68f` | `--success` |
| 错误（答错） | `#ff6b6b` | `--danger` |
| 警告（计时器） | `#ffa726` | `--warning` |
| 边框 | `rgba(255,255,255,0.1)` | `--border` |

### 渐变装饰

- 背景径向渐变：紫色（20% 左中）+ 绿色（80% 右上），透明度 0.08~0.1
- 标题文字渐变：`#6c5ce7` → `#00d68f`
- 按钮渐变：`#6c5ce7` → `#5a4bd1`

## 排版

| 层级 | 字号 | 粗细 | 用途 |
|------|------|------|------|
| 页面标题 | 28px | 700 | `.title` |
| 段落标题 | 22px | 700 | `.page-title` |
| 正文 | 14–16px | 400 | 界面文字 |
| 小字 | 12–13px | 400 | 辅助说明 |
| Emoji 展示 | 56px | — | `.emoji-display` |
| 数据数字 | 28px | 700 | `.stat-value` |

字体栈：`-apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif`

## 圆角与间距

| 元素 | 圆角 | 说明 |
|------|------|------|
| 卡片 | `16px` | `--radius` |
| 按钮/输入框 | `10px` | `--radius-sm` |
| 反馈条/提示 | `10px` | `--radius-sm` |

按钮内边距：主按钮 `14px 20px`，小按钮 `8px 16px`

卡片内边距：`36px 28px`（桌面），`24px 18px`（移动端 `<400px`）

## 动画规范

| 动画 | 时长 | 缓动 | 用途 |
|------|------|------|------|
| fadeInUp | 0.4s | ease | 页面/卡片/反馈出现 |
| bounce | 2s 循环 | ease | 首页 logo 浮动 |
| shake | 0.5s | ease | 答错时 emoji 抖动 |
| pop | 0.4s | ease | 答对时 emoji 放大弹跳 |
| pulse | 0.5s | ease | combo 数字跳动 |
| 按钮 active | 0.2s | — | 按下缩放 `scale(0.97)` |

## 组件规范

### 按钮层级
1. **主按钮** `.btn-primary`：渐变紫底，白色文字，全宽
2. **描边按钮** `.btn-outline`：透明底 + 白色边框，全宽
3. **小描边按钮** `.btn-outline-small`：透明底 + 灰色文字，自适应宽度

### 反馈状态
- **答对**：绿色背景 `rgba(0,214,143,0.12)` + 绿色边框 + 绿色文字
- **答错**：红色背景 `rgba(255,107,107,0.12)` + 红色边框 + 红色文字

### 计时器
- 正常：`--success` 绿色
- ≤10 秒：`--warning` 黄色
- ≤5 秒：`--danger` 红色

## 响应式断点

| 断点 | 适配内容 |
|------|----------|
| > 400px | 标准布局，卡片 max-width 480px |
| ≤ 400px | 缩小内边距、emoji 字号 42px、数据数字 22px |

## 可访问性

- 输入框聚焦有紫色光晕（`box-shadow`），便于键盘用户感知
- 按钮 disabled 状态视觉明确（灰色 + 禁止点击光标）
- 颜色不是唯一的信息传达方式（正确/错误同时有文字 + emoji 图标 + 颜色）
