# 项目需求文档 — AI 表情猜谜王

> 版本：v1.0 | 更新：2026-06-02 | 状态：开发中

## 项目概述

一款纯前端 + 简易后端的网页游戏。AI 动态生成 emoji 谜题，玩家看 emoji 猜词语/成语/电影名。支持计分、难度关卡、本地排行榜。

## 目标用户

- 任何喜欢猜谜、emoji 文化的玩家
- 手机和电脑均可游玩

## 功能需求

### F1 — 难度选择
- 玩家从三个难度中选择：简单 / 中等 / 困难
- 不同难度影响题目复杂度、emoji 数量、是否有提示、是否限时

### F2 — AI 动态出题
- 后端调用 AI API（Anthropic / OpenAI）实时生成谜题
- AI 返回 JSON 格式：`{ emoji, answer, hint }`
- AI 不可用时自动 fallback 到本地预设题库（30 题）

### F3 — 猜词玩法
- 玩家输入文字答案，提交后判定对错
- 判定规则：大小写不敏感、去除首尾空格
- 答对/答错有视觉反馈动画
- 支持"跳过"按钮

### F4 — 计分系统
- 答对加分：简单 +10 / 中等 +20 / 困难 +30
- 连续答对 combo 加成：额外 +2 分/连击（最高 +10）
- 答错或跳过：combo 中断
- 每轮 5 题

### F5 — 计时器（困难模式专属）
- 困难模式每题限时 30 秒
- 进度条可视化倒计时，最后 10 秒变黄，最后 5 秒变红
- 超时算答错

### F6 — 结算页
- 展示总得分、答对题数、最高连击
- 根据正确率给出不同评语和 emoji
- 自动保存成绩到排行榜

### F7 — 本地排行榜
- localStorage 存储，Top 20
- 显示排名、昵称、分数、难度、日期
- 前三名金银铜标识

### F8 — 昵称系统
- 游戏开始前输入昵称（最长 10 字）
- 昵称为空时不可开始游戏

## 非功能需求

| 需求 | 说明 |
|------|------|
| 响应式 | 手机和桌面端均可使用 |
| 性能 | 页面加载 < 2s，API 响应 < 5s |
| 安全 | API Key 仅存后端 .env，不暴露给前端 |
| 兼容 | Chrome / Safari / Firefox 最新版 |
| 离线兜底 | 后端挂掉时预设题库仍可玩 |

## 页面流程

```
[开始页] → [输入昵称] → [选择难度] → [游戏循环 5 题] → [结算页] → [排行榜]
    ↑                                                        │              │
    └────────────────────────────────────────────────────────┴──────────────┘
```
